# 题库系统 - 微信小程序

## 📱 项目概述
这是题库系统的微信小程序版本，支持：
- 题目浏览和搜索
- 智能对话查询
- 错题本管理
- 学习进度跟踪

## 🏗️ 项目结构
```
wechat-miniprogram/
├── app.js              # 小程序入口
├── app.json            # 小程序配置
├── app.wxss            # 全局样式
├── pages/              # 页面目录
│   ├── index/          # 首页
│   ├── questions/      # 题目列表
│   ├── search/         # 搜索页面
│   ├── chat/           # 对话页面
│   └── profile/        # 个人中心
├── components/         # 公共组件
├── utils/              # 工具函数
├── config/             # 配置文件
└── assets/             # 静态资源
```

## 🔧 开发环境
1. 下载微信开发者工具
2. 注册微信小程序账号
3. 导入本项目
4. 配置服务器域名

## 🔗 API接口
- 基础URL: https://yourdomain.com
- 接口文档: https://yourdomain.com/docs

## 📦 功能模块
- [ ] 用户登录（微信授权）
- [ ] 题目浏览和搜索
- [ ] 智能对话助手
- [ ] 错题本功能
- [ ] 学习统计
- [ ] 消息通知

## 🚀 快速开始
1. 修改 `config/api.js` 中的服务器地址
2. 在微信公众平台配置服务器域名
3. 使用微信开发者工具打开项目
4. 点击预览或上传
