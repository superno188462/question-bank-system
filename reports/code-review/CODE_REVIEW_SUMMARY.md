## 🔴 必须修改（安全问题）

### 1. API Key 明文存储（严重）
- **位置**: `config/agent.json`
- **风险**: API Key 可能泄露到 Git 仓库
- **修复**: 
  - 将 `config/agent.json` 添加到 `.gitignore`
  - 使用环境变量存储 API Key
  - 创建 `agent.json.example` 作为模板

### 2. 输入验证不足（严重）
- **位置**: AI 相关 API 接口
- **风险**: 恶意输入可能导致服务滥用
- **修复**: 添加严格的 Pydantic 验证器

### 3. CORS 配置过于宽松
- **位置**: `web/main.py`
- **风险**: 允许任意来源访问 API
- **修复**: 生产环境限制具体来源
